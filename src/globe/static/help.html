Help & troubleshooting

Quick answers for hosts and demo visitors.

For demo visitors

  • Open the console and follow the Getting started checklist on Overview.
  • Click Run demo flow for an automated walkthrough.
  • Leave the API key blank unless your host gave you one.
  • If buttons do nothing, refresh the page and check your connection.

For hosts sharing a demo

  • Run ./scripts/setup.sh once, then ./scripts/share-demo.sh.
  • Share the /app URL printed when the tunnel starts.
  • Keep your machine awake while the tunnel is running.
  • Verify with ./scripts/smoke-test.sh before sharing.

Common issues

  • 401 / invalid API key — Click Clear key in Settings and refresh.
  • Cannot reach API — Ensure GlobeCloud is running (./scripts/start-clean.sh).
  • Slow console — Kill zombie on port 8001: ./scripts/start-clean.sh.
  • Connection refused (tunnel) — Restart the server, then restart the tunnel.
